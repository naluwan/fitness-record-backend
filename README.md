# Fitness Record

## 啟動專案

### 下載專案
```
$ git clone https://github.com/naluwan/fitness-record-backend.git
```

### 安裝依賴
```
npm install
```
### 啟動

```
$ npm run dev
```

### 終端機提示

```
fitnessRecord app listening on http://localhost:3000
```

### 瀏覽網址

```
http://localhost:3000
```
